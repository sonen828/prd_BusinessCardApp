# Antigravity 設定の自動生成

1. ユーザーのリクエストを分析し、**ルール**（受動的な指針）、**ワークフロー**（能動的な手順）、または **スキル**（能力拡張）かを判断する。
2. **config-expert スキル** を参照して、正しいフォーマットとベストプラクティスを確認する。
// turbo
3. `config-expert` の基準に従い、新しいファイルの内容を作成する。
4. 適切なディレクトリにファイルを作成する：
    - ルール: `.agent/rules/`
    - ワークフロー: `.agent/workflows/`
    - スキル: `.agent/skills/<skill-name>/`
5. 作成したファイルの場所をユーザーに通知する。
